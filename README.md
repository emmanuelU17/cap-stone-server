# Sara Brand Server
Server side code for Sara Brand e-commerce. Will become a private repo on completion of product relationship

### Technologies Thus Far
* Java 17
* Spring Boot
* MySQL and Redis
* Flyway Migration
* Unit test JUnit and Mockito
* Integration test using test containers
