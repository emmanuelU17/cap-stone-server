# Sarre The Brand Server
Server side code for Sarre The Brand.

### Technologies
* Java 17
* Spring Boot
* Flyway Migration
* AWS RDS (MySQL)
* AWS Secret manager
* AWS EC2
* AWS SES
* Unit test JUnit and Mockito
* Integration test using test containers
